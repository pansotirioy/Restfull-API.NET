# RESTful API with Entity Framework Core – Degree Certification System

## Overview

This project is a **backend-only RESTful API** developed using **ASP.NET Core** and **Entity Framework Core**. The system is designed to manage and verify **academic degree certifications** for an organization. It provides structured API endpoints that allow external applications or services to interact with the certification database.

The API enables efficient management of degree records, including storing graduate information, degree titles, and certification status. It follows REST principles to ensure scalability, maintainability, and easy integration with frontend applications or other services.

## Features

* Manage degree certification records
* Create, retrieve, update, and delete certification data (CRUD operations)
* Structured RESTful architecture
* Database interaction using Entity Framework Core
* Clean backend architecture for maintainability and scalability
* Designed for integration with external systems or frontend clients

## Technologies Used

* **ASP.NET Core Web API**
* **Entity Framework Core**
* **RESTful Architecture**
* **C#**
* **Relational Database (SQL-based)**

## Database Structure

The database is designed to store and manage certification information such as:

* Graduate / student details
* Degree title
* Issuing organization
* Certification status
* Date of issuance

Entity Framework Core is used as the **ORM** to map the application models to database tables and manage data access efficiently.

## Purpose

The goal of this project is to demonstrate the implementation of a **RESTful backend service** using **ASP.NET Core and EF Core**, focusing on clean architecture, structured data handling, and scalable backend design.
